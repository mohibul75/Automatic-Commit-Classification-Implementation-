# Automatic-Commit-Classification-Implementation-
This repository is about the Implementation  of "Boosting Automatic Commit Classification Into Maintenance Activities By Utilizing Source Code Changes".

**********
<h3>Implementation</h3>

The full process of implementation is given bellow :

<ol>

<li>Download data from provided link</li>
<li>Preprocess the data (remove the unnecessary row and colum)</li>
<li>
<ul>
<li> Prepare dataset for keyword model</li>
<li>Prepare dataset for combained(keyword + ) model</li>
</ul>
</li>
<li>Split dataset (85/15)</li>
<li> Define the model</li>
<li>Train the model with 85% data</li>
<li>Save the model</li>
<li>Loada and Test with rest of the 15% data.</li>
</ol>
